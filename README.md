# 0 整体要求
- 简约、实用、舒适，不要求特定风格（类 日式+北欧）, 有日式的禅意
- 主色调 白色 灰色 木色 等亮色
- 要求硬装尽量简单 质量好
- 收纳要多
- 插座 要足够
- 灯的开关 需要双控开关：卧室、客厅（沙发上方有射灯），餐桌有灯，厨房操作台灯
- 准备养一只金毛，一只猫，猫窝狗窝优先放大阳台
- 供暖：水地暖
- 空调
- 地板 木纹瓷砖 、浅色瓷砖 或者 多层、三层复合地板
- 不安装防盗窗，可能安装**金刚网**
- 普通的腻子，可选用用**耐水腻子**



## 可选
- 门 [丁盯安全门锁](https://item.jd.com/10095353020.html)
- 挂自行车

# 4 核心区 - 动： 客厅
